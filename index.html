00Logo · 标志系统4 个变体 · 一物一用 · 别混用

每个变体有 固定使用场景，选错=品牌识别度打折。优先选最贴合载体的那一个，不要把横版强塞进 App icon 槽位、也不要把全身版用在 16px favicon。

Horizontal · 主横版 PRIMARY
用在：网站头部 / 文档头 / 邮件签名 / PPT 封面 / 印刷物料 / 媒体露出
最小宽度：160px（再小读不清 mascot 细节）
文件：assets/logo/horizontal.png
Compact · 横版紧凑（中英堆叠） UI
用在：导航栏 logo 槽位 / Footer / 移动端 header / 名片 / 邀请函
最小宽度：120px
文件：assets/logo/compact.png
App Icon · 圆角方形容器 APP / PWA
用在：iOS / Android app icon / PWA 桌面图标 / favicon (256+)
容器：Coral 渐变方形 + 28% 圆角 + 内嵌 mascot 头
文件：assets/logo/app-icon.png
Full Body · 全身 + 文字 MARKETING
用在：营销 banner / Hero illustration / 404 / Empty state / 社媒头图
最小高度：240px（要看清 hoodie 细节）
文件：assets/logo/fullbody.png
Avatar · 圆头像 / Favicon FAVICON
用在：App 头像位 / Favicon / 收藏卡 chip / 内嵌行内小图标
最小尺寸：16×16（favicon）/ 32×32（app）
文件：assets/logo/avatar.png
Lockup Mono · 黑白单色 ⭐ NEW MONO
用在：打印 / T-shirt / 灰底 / 反白场景（深色背景需反白处理）/ 任何不允许彩色 IP 露出的合作物料
一句话规则：有 Coral 配色用 horizontal，没有用 lockup-mono
文件：assets/logo/lockup-mono.png
CLEAR SPACE · 安全间距
Logo 周围保留 ≥ mascot 头高度的 50% 留白，禁止文字 / 图形挤压。
背景使用规则
✅ Warm Canvas #FFFEF7
✅ White
❌ 高饱和品牌色背景（识别度差）
❌ 暗色背景（mascot 主体是黑，会糊）
📌 暗色背景版 logo（白色 mascot 描边版）暂未做，需要时让设计师补一版反白。

19Do & Don't · 红线PR 看到 Don't 立即打回

✅ Do

  • 主背景永远 #FFFEF7 Warm Canvas
  • 按钮永远 Full Pill + 渐变 + Glow
  • Mascot 9 态对号入座
  • 卡片默认 24px 圆角
  • 一屏品牌瞬间 1 个就够（mascot/sticker/squiggle/blob）
  • 文案"人话 + 推力"
  • 游戏化做进度和完成态

❌ Don't

  • 主背景用纯白 #FFFFFF
  • 矩形 / 小圆角按钮、扁平蓝色按钮
  • 蓝色作为品牌主识别色（这是 SaaS 味）
  • 暗色背景 + 发光特效堆品牌感
  • 一屏 ≥3 个强渐变 / ≥3 个 sticker
  • 用 Ant Design / MUI 默认外观
  • Mascot 状态贴错（错误页贴 Happy）
  • Caveat 字体进正文 / 标题
